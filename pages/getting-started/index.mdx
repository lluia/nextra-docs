# About Auth.js

Auth.js is a complete open-source authentication solution for web applications.<br />Check out the live demos of Auth.js in action:

<DemoCards />

Support for more frameworks are on the way, check the [integrations page](/getting-started/integrations)

## What is Auth.js?

Auth.js is a runtime agnostic authentication library ecosystem based on standard Web APIs that integrates with modern JavaScript frameworks.

<Callout type="info">
  To reduce migration overhead, NextAuth.js will continue to be released under the `next-auth` package name, instead of the
  `@auth/*` convention. Auth.js was born out of `next-auth` but [over
  the years has evolved](/getting-started/contributors#history) to be framework
  agnostic.
</Callout>

## Authentication methods

There are 3 ways to authenticate users with Auth.js:

- [OAuth authentication](/getting-started/providers/oauth-tutorial) ( _Sign in with Google, Github, LinkedIn, etc..._ )
- [Token-based authentication](/getting-started/providers/email-tutorial) ( _Magic links, OTP codes, etc..._ )
- [Credentials](/getting-started/providers/credentials-tutorial) ( _Integrating with external APIs,
  WebAuthn, etc..._ )
